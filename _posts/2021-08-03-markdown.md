# Markdown Examples

## 1. Headings

```
# Heading Example
## Heading Example
### Heading Example
```

## 2. Italic, Bold, etc..

Italic = `*example*` or `_example_` <br>
Bold = `**example**` or `__example__` <br>
Blockquote = `>` or `>>` <br>
Horizontal Lines = `---` or `***` <br>
